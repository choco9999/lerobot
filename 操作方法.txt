[強化学習(HIL-SERL)]
# 事前に依存関係を入れる（未導入の場合）
#   pip install -e ".[hilserl]"
#
# 1) 設定を編集（ポート/カメラ/関節数/ action の min/max）
#   src/lerobot/configs/train_config_hilserl_so101.json
#
# 2) learner を起動（別ターミナル）
#   python -m src.lerobot.rl.learner --config_path src/lerobot/configs/train_config_hilserl_so101_act.json
#
# 3) actor を起動（別ターミナル）
#   python -m src.lerobot.rl.actor --config_path src/lerobot/configs/train_config_hilserl_so101_act.json
#
# NOTE:
# - ckpt（チェックポイント）は learner が `outputs/train/.../checkpoints/last/` に保存する（actor 側には作られない）
# - `output_dir` が `null` だと learner/actor で別フォルダになりやすいので、必要なら両方に同じ `--output_dir=...` を指定する
# - `save_freq` は optimization step 単位（環境ステップではない）。早く ckpt が欲しい場合は `save_freq` を下げる
# - `save_replay_buffer_dataset=true` は `<output_dir>/dataset` へ書き出す（実機だと重くなりがちなので通常は false 推奨）
#
# 介入・ラベル操作（SO101 leader 使用時）
#   space : 介入ON/OFF（トグル）
#   s     : 成功（success）
#   q/esc : 失敗（terminate）
#   r     : rerecord（terminate + rerecord）
